php7编译安装mongodb扩展

PHP

关于扩展安装这块的步骤，单纯记录，大佬忽略。

首先要下载mongodb的php扩展。

wget https://pecl.php.net/get/mongodb-1.1.1.tgz  

// 接着解压并进入该文件夹 

tar -zxvf ./mongodb-1.1.1.tgz && cd mongodb-1.1.1

接着就直接和其他的编译扩展是一个类型了。

// 路径自己替换一下，找到自己的phpize 和php-config就好

/usr/local/php7/bin/phpize && ./configure --with-php-config=/usr/local/php7/bin/php-config

// 然后把生成的mongodb.so复制到相应的位置。在php.ini中写入

extension = mongodb.so;

然后重启php-fpm。访问php_info，显示已经安装了相应的扩展。
