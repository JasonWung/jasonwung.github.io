Ubuntu16.04 安装ftp

Ubuntu16.04 安装ftp，根据依赖安装。

安装vsftpd。

sudo apt-get install vsftpd

结束后则安装完毕。

配置vsftpd。

// 打开编辑文件
sudo vi /etc/vsftpd.conf

// 设置本地账户可以以ftp账户登录
local_enable=YES

// 使用用户的本地账户目录作为ftp目录 
chroot_local_user=YES 

// 允许用户上传文件
write_enable=YES 

// 修改完成后保存并重启vsftpd 
sudo /etc/init.d/vsftpd restart 

// 添加ftp使用用户
sudo useradd -d /home/ftp_testuser -m ftp_testuser 
// 设置账户密码
sudo passwd ftp_testuser
